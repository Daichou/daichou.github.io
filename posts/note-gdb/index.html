[Note] GDB 一些指令功能速查

前情提要

這篇主要是介紹gdb的一些常用功能，遇到的時候方便google。

常用功能

  • backtrace
  • breakpoint
  • watchpoint
  • stopwatch
  • dump register
  • dump variable
  • continue and stop
  • disasmmebly
  • return

指令速查

  • backtrace
    • bt
  • breakpoint
    • b
    • br
  • watchpoint
    • watch variable_name or watch variable_address
  • stopwatch
    • maint link
  • dump register
    • info r
  • dump variable
    • print variable_name
    • print &variable_name => get address
    • print *variable_address => get address contents
    • info display
  • continue and stop
    • c => continue
    • r= > run
    • stop => Ctrl+c
  • disasmmebly
    • disassemble (func…) link
    • layout asm
    • layout src
  • return
    • return (return this stack)
